# Smart Water Level Monitoring IoT-based System (with ESP32)

Given in the repository is the code required to construct a smart water level monitoring IoT system, that utilizes the WiFi + MQTT (HTTP) protocol connectivity of the ESP32 microcontroller to transmit data received from multiple sensors to a ThingSpeak server and display it in the form of a graph. The system measures the water level in a bottle or glass container using an ultrasonic sensor and calibrates the measurements using a water level sensor as a reference. The ESP32 processes sensor data and transmits the information to a remote system (here, a ThingSpeak server) using HTTP communication. An IR sensor is used to detect the presence of the container during the experiment setup. Thus, the given setup demonstrates how sensor calibration can improve measurement accuracy, and hot IoT communication allows for the transfer of remote data.
